# КИМ Самостоятельная работа

Репозиторий содержит контрольно-измерительные материалы (КИМ) для дисциплины «Управление проектами в сфере ИИ».

## Состав

**[KIM_SR](KIM_SR/README.md)** — сквозной КИМ для самостоятельной работы (СР): аналитическое эссе по темам, вынесенным в РПД на самостоятельное изучение (Модули 1, 2, 3, 4, 6, 7, 8). 

## Источник содержания

Материалы подготовлены на основе РПД дисциплины «Управление проектами в сфере ИИ» и адаптированы к логике ФОС.
