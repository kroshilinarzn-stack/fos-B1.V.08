# M3. Управление человеческими ресурсами проекта (команда)

Папка содержит образец полного КИМ для модуля 3 дисциплины «Управление проектами в сфере ИИ».

Назначение образца — показать единый формат оформления модульного КИМ в репозитории ФОС: паспорт, проверяемые результаты, ход выполнения, формат сдачи, рубрика, контрольные вопросы, использование ИИ-инструментов и рекомендации преподавателю.

## Состав

- `task.md` — полный текст КИМ для обучающихся;
- `rubric.md` — критерии оценивания и шкала 4/6/8/10;
- `student_report_template.md` — шаблон отчёта обучающегося;
- `teacher_notes.md` — методические указания для преподавателя;
- `variants.md` — варианты индивидуализации и усложнения задания.

## Источник содержания

Образец подготовлен на основе рабочей программы дисциплины «Управление проектами в сфере ИИ» (раздел «Модуль 3. Управление человеческими ресурсами проекта (команда)»: структура команды, ответственность, конфликты; лидерство и самоорганизация в R&D-командах; кросс-функциональные команды в высокотехнологичных проектах; структура и RACI для AI-команды) и адаптирован к логике ФОС.
