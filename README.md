# Фонд оценочных средств (ФОС)
## Дисциплина
**Управление проектами в сфере ИИ**
## Назначение
Данный репозиторий содержит фонд оценочных средств (ФОС) по дисциплине **«Управление проектами в сфере ИИ»**.
ФОС предназначен для организации, проведения и документирования процедур оценки результатов обучения обучающихся в соответствии с рабочей программой дисциплины и требованиями образовательной программы.
Репозиторий включает:
- паспорт ФОС;
- матрицу соответствия компетенций и оценочных средств;
- критерии оценки работ студентов;
- шаблоны оценочных листов;
- материалы для валидации и сопровождения ФОС;
- документацию по применению ФОС.
## Целевая аудитория
Репозиторий предназначен для:
- преподавателей дисциплины;
- студентов;
- разработчиков фонда оценочных средств;
- методистов образовательной программы.
## Структура репозитория
Основные разделы репозитория:
- **meta/** — исходные данные и методические решения.
- **docs/** — документация ФОС.
- **data/** — карты компетенций и машиночитаемые данные.
- **src/** — рубрики оценивания и шаблоны документов.
- **tests/** — материалы проверки полноты и согласованности ФОС.
- **validation/** — материалы пилотной валидации.
- **release/** — инструкции по применению ФОС и печатные материалы.
## Быстрый старт
1. Ознакомьтесь с документом **`FOS_BRIEF.md
`**, содержащим краткое описание фонда оценочных средств.
2. Изучите паспорт ФОС (**`docs/passport.md`**) и архитектуру оценивания (**`docs/fos_architecture.md`**).
3. Просмотрите критерии и рубрики оценивания в каталоге **`src/rubrics/`**.
4. Используйте шаблоны оценочных листов из каталога **`src/templates/`** при проведении оценивания.
5. Для внедрения ФОС в образовательный процесс следуйте инструкции **`release/how_to_apply.md`**.
## Лицензия
Материалы репозитория предназначены для использования в образовательном процессе в соответствии с локальными нормативными документами образовательной организации.

## 📌 О курсе

Дисциплина «Управление проектами в сфере ИИ» входит в цикл профессиональных модулей программы дополнительного профессионального образования (ДПО). Курс ориентирован на формирование у слушателей системного подхода к планированию, организации и контролю проектов по разработке и внедрению искусственного интеллекта.
