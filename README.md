# Фонд оценочных средств по дисциплине «Управление проектами в сфере ИИ»

Этот репозиторий предназначен для открытого и воспроизводимого представления фонда оценочных средств (ФОС) по дисциплине «Управление проектами в сфере ИИ». Здесь объединены рабочая программа дисциплины, модель измерения результатов обучения, контрольно-измерительные материалы (КИМ), критерии оценивания, методические рекомендации и используемые образовательные ресурсы. Результаты обучения связываются с компетентностно-ролевой моделью в области искусственного интеллекта (КРМ версии 3.0).

## 1. О дисциплине

**Цель дисциплины -** сформировать компетенции по управлению ИИ-проектами на всех этапах жизненного цикла, включая командную работу и применение ИИ-инструментов для оптимизации управления

**Основные задачи.** 
1.	Сформировать знания о ключевых принципах и методах управления проектами и научить адаптировать их под специфику разработки ПО и систем ИИ.
2.	Развить навыки применения ИИ-инструментов проектного планирования и управления требованиями проектной работы.
3.	Научить обосновывать экономическую целесообразность внедрения ИИ-решений, оценивать трудоёмкость, сроки и стоимость ИИ-задач.
4.	Развить навыки выявления и управления рисками ИТ- и ИИ-проектов.
5.	Сформировать компетенции по созданию проектной команды и управлению ею на всех стадиях жизненного цикла ИИ-проекта.

**Профильные роли по КРМ**

| **Роль** | **Статус** | **Функция в дисциплине и проекте** |
| :--- | :--- | :--- |

| **AI CDO** (Руководитель цифровой трансформации) | поддерживающая | проектирование новых путей формирования ценности в организации посредством применения ИИ, стратегическое планирование, управление изменениями и формирование портфеля ИИ-проектов |

**Структура и содержание.** 

Дисциплина состоит из **8 модулей:**

  **Модуль 1. Проекты и среда управления**.
  
  **Модуль 2. Управление содержанием проекта**.
  
  **Модуль 3. Управление человеческими ресурсами проекта (команда)**.
  
  **Модуль 4. Управление сроками проекта**.
  
  **Модуль 5. Управление стоимостью проекта**.
  
  **Модуль 6. Управление качеством проекта**.
  
  **Модуль 7. Управление рисками и взаимодействием в проекте**.
  
  **Модуль 8. Управление проектами в сфере искусственного интеллекта: методы, подходы, инструментарий**.


Обучение включает [лекции / практические занятия / лабораторные работы / проект / самостоятельную работу] и завершается [экзаменом в виде командной защиты ИИ-проекта (в форме питча)

## 6. Порядок заполнения

1. Заполните РПД и сформулируйте проверяемые результаты обучения.
2. Выберите релевантные профессиональные роли и компетенции КРМ.
3. Для каждого результата выберите индикатор и требуемый уровень.
4. Заполните модель измерения.
5. Создайте КИМ и рубрики оценивания по шаблонам.
6. Согласуйте систему итогового балла.
7. Добавьте ресурсы, сведения о команде и лицензию.
