# Рубрика оценивания практического занятия «Управление стейкхолдерами»

Оценивание проводится по единой 10-балльной шкале ФОС. Перевод результата в БРС выполняется не умножением на 10, а по весу данной работы, который фиксируется в `docs/evaluation_policy.md`.

## 1. Критерии оценивания

| Критерий | Баллы |
|---|---:|
| Полнота идентификации стейкхолдеров (внутренние и внешние группы) | 1.5 |
| Корректность сбора и анализа ожиданий стейкхолдеров (выявление конфликтов интересов) | 1.5 |
| Обоснованность классификации стейкхолдеров по матрице «влияние — интерес» | 2.0 |
| Качество разработки стратегий взаимодействия для каждой группы стейкхолдеров | 1.5 |
| Полнота и реалистичность плана коммуникаций (форматы, периодичность, ответственные) | 1.5 |
| Оформление отчёта, качество защиты и обоснованность выводов | 2.0 |
| **Итого** | **10.0** |
