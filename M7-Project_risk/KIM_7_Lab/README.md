# Реестр рисков TOP-10
Папка содержит комплект документов для лабораторной работы по дисциплине «Управление проектами», продолжающей цикл работ по управлению AI-проектом: построение матрицы рисков, ресстр рисков на основе перечня работ, критического пути, сформированных в предыдущих лабораторных работах.

## Состав

- [kim_7_lab_task.md](kim_7_lab_task.md) — полный текст лабораторной работы для обучающихся;
- [kim_7_lab_mark.md](kim_7_lab_mark.md) — критерии оценивания;
- [kim_7_lab_student_report_template.md](kim_7_lab_template.md) — шаблон отчёта обучающегося;
- [kim_7_lab_teacher_notes.md](kim_7_lab_recom.md) — методические указания для преподавателя;
- [kim_7_lab_variants.md](kim_7_lab_var.md) — варианты индивидуализации задания.

## Источник содержания

Комплект подготовлен на основе методики управления рисками (Pre-morten анализе, сценарного моделирования "Что, если?", матрицы рисков) и адаптирован к логике единого пакета ФОС, использованной в предыдущих работах цикла: терминология приведена к единому виду с приставкой «AI-» (AI-проект, AI-команда, AI-роли), а исходными данными служит перечень работ, сформированный в лабораторной работе «Формирование AI-команды, рабочего пространства и перечня работ AI-проекта».
