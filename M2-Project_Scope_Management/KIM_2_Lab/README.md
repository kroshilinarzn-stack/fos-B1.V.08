# M2. Управление содержанием проекта

Папка содержит контрольно-измерительный материал (КИМ_2_Lab) для второй части модуля 2 дисциплины «Управление проектами в сфере ИИ».

КИМ содержит: паспорт, проверяемые результаты, ход выполнения, формат сдачи, оценку результатов, контрольные вопросы, использование ИИ-инструментов и рекомендации преподавателю.

## Вид КИМ: лабораторная работа

## Состав

- **kim_2_Lab_task.md** — полный текст КИМ для обучающихся;
- **kim_2_Lab_mark.md** — критерии оценивания;
- **kim_2_Lab_template.md** — шаблон отчёта обучающегося;
- **kim_2_Lab_recom.md** — методические указания для преподавателя;
- **kim_2_Lab_var.md** — варианты индивидуализации задания.

## Источник содержания

Образец подготовлен на основе исходного теоретического и практического материала модуля 2 в рамках темы «Разработка устава и содержания AI-проекта» и адаптирован к логике ФОС.
