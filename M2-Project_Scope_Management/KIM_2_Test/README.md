# M2. Управление содержанием проекта

Папка содержит контрольно-измерительный материал (КИМ_2_Test) для первой части модуля 2 дисциплины  
«Управление проектами в сфере ИИ».

КИМ содержит: паспорт, проверяемые результаты, источники информации, рекомендации по настройке теста и условиям тестирования, оценку результатов, а также банк тестовых вопросов по следующим темам дисциплины:

1. Определение целей и содержания проекта, инструменты Impact Mapping, Customer Journey Mapping, User Story Mapping, MBO, разработка WBS, техника SWOT-анализа, «Шесть сигм».  
2. Современные подходы к управлению содержанием сложных технологических проектов.  
3. Разработка устава AI-проекта, анализ стейкхолдеров и требований; применение Impact/User Story Mapping и WBS для определения содержания и границ проекта.

## Вид КИМ: тест

## Состав

- Question_bank — содержит XML-файлы со списками тестовых вопросов, разделённые на 10 групп (`kim_1.1_gr1.xml` – `kim_1.1_gr10.xml`);
- kim_2_test_prompt.md — промпт для генерации тестовых вопросов;
- kim_2_test_task.md — описание формирования и настройки теста; процедуры тестирования и оценки результатов;
- kim_2_test_mark.md — критерии оценивания;
- kim_2_test_recom.md — методические указания для преподавателя;
- kim_2_test_var.md — варианты индивидуализации задания.

## Источник содержания

Образец подготовлен на основе исходного теоретического и практического материала модуля 2 в рамках тем «Инструменты целеполагания и декомпозиции в проектах; управление содержанием технологических проектов; разработка устава и содержания AI-проекта».
