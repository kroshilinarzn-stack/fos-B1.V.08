# M2. Управление содержанием проекта

Папка содержит контрольно-измерительные материалы для модуля 2
дисциплины «Управление проектами в сфере ИИ».

## Состав

- **[KIM_2_Test](KIM_2_Test/README.md)** — контрольно-измерительные материалы (**в виде теста**) по первой части модуля 2, включающей следующие темы:  
  - Определение целей и содержания проекта;
  - Инструменты Impact Mapping, Customer Journey Mapping, User Story Mapping, MBO, разработка WBS, техника SWOT-анализа, «Шесть сигм»;
  - Современные подходы к управлению содержанием сложных технологических проектов.

- **[KIM_2_Lab](KIM_2_Lab/README.md)** — контрольно-измерительные материалы (**в виде лабораторной работы**) по второй части модуля 2, включающей следующие темы:  
  - Разработка устава AI-проекта;
  - Применение Impact Mapping, User Story Mapping и WBS для определения содержания и границ проекта.

- **[KIM_2_Pract](KIM_2_Pract/README.md)** — контрольно-измерительные материалы (**в виде практического задания**) по третьей части модуля 2, включающей следующие темы:  
  - Анализ производственной задачи предприятия, определение целесообразности применения ИИ, постановка целей проекта;
  - Выявление требований заказчика, определение границ проекта, анализ стейкхолдеров, выбор состава работ.

## Источник содержания

Образец подготовлен на основе рабочей программы дисциплины «Управление проектами в сфере ИИ» (раздел "Модуль 2. Управление содержанием проекта": инструменты целеполагания и декомпозиции в проектах; управление содержанием технологических проектов; разработка устава и содержания AI-проекта) и адаптирован к логике ФОС.
