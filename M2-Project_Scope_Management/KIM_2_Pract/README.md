# M2. Управление содержанием проекта

Папка содержит контрольно-измерительный материал (КИМ_2_Pract) для третьей части модуля 2 дисциплины «Управление проектами в сфере ИИ».

КИМ содержит: паспорт, проверяемые результаты, ход выполнения, формат сдачи, оценку результатов, контрольные вопросы, использование ИИ-инструментов и рекомендации преподавателю.

## Вид КИМ: практическое задание

## Состав

- **[kim_2_pract_task.md](kim_2_pract_task.md)** — полный текст КИМ для обучающихся;
- **[kim_2_pract_mark.md](kim_2_pract_mark.md)** — критерии оценивания;
- **[kim_2_pract_template.md](kim_2_pract_template.md)** — шаблон отчёта обучающегося;
- **[kim_2_pract_recom.md](kim_2_pract_recom.md)** — методические указания для преподавателя;
- **[kim_2_pract_var.md](kim_2_pract_var.md)** — варианты индивидуализации задания.

Образец подготовлен на основе исходного теоретического и практического материала модуля 2 в рамках темы «Управление содержанием AI-проекта и анализ требований» и адаптирован к логике ФОС.
