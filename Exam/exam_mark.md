# КРИТЕРИИ ОЦЕНИВАНИЯ СКВОЗНОГО ПРОЕКТА
## Дисциплина «Управление проектами в сфере искусственного интеллекта»

Общая максимальная оценка за проект составляет **10 баллов**.

### 1. Распределение баллов по критериям

| Критерий оценивания | Макс. балл |
| :--- | :---: |
| Полнота комплекта проектной документации | 1,5 |
| Корректность применения методов управления проектами | 1,5 |
| Качество разработки календарного плана, сетевой модели и бюджета | 1,5 |
| Обоснованность решений по качеству, рискам и ресурсам | 1,5 |
| Логическая связность проекта и согласованность документов | 1,0 |
| Практическая реализуемость AI-проекта | 1,0 |
| Качество оформления документации | 0,5 |
| Качество презентации проекта (питча) | 0,5 |
| Ответы на вопросы комиссии | 1,0 |
| Личный вклад обучающегося | 1,0 |
| **ИТОГО** | **10,0** |

---

### 2. Уровни оценивания (Шкала)

* **Отлично (9,0 – 10,0 баллов)**
  Обучающийся демонстрирует сформированность компетенций на высоком уровне. Все документы логически согласованы, расчеты корректны, проект реализуем, ответы на вопросы полные и аргументированные.
  
* **Хорошо (7,5 – 8,9 баллов)**
  Допускаются отдельные неточности, не влияющие на общую работоспособность проекта и логику управления.
  
* **Удовлетворительно (6,0 – 7,4 баллов)**
  Проект выполнен полностью, однако имеются методические ошибки, недостаточная глубина анализа и неполное обоснование принятых решений.
  
* **Неудовлетворительно (менее 6,0 баллов)**
  Проект содержит существенные ошибки, отсутствуют обязательные разделы, обучающийся не может обосновать принятые решения.

---

### 3. Основания для снижения оценки (Штрафные санкции)

Оценка может быть снижена при наличии следующих недостатков:

| Выявленный недостаток | Размер снижения |
| :--- | :---: |
| Отсутствует один обязательный раздел проекта | до **1 балла** |
| Обнаружены существенные противоречия между документами | до **1 балла** |
| Некорректно построена WBS, сетевая модель или диаграмма Ганта | до **1 балла** |
| Необоснован бюджет проекта | до **1 балла** |
| Формальный анализ рисков или качества | до **0,5 балла** |
| Не подтверждён личный вклад обучающегося | до **2 баллов** |
| Неспособность ответить на вопросы по собственной части проекта | до **2 баллов** |

*Примечание: Снижения по разным критериям суммируются. При снижении итоговой оценки ниже 6 баллов проект признается не сданным.*
