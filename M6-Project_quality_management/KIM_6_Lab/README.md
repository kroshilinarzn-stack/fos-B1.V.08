# КИМ 6. Управление качеством проекта

Папка содержит комплект документов для лабораторной работы с элементами коллоквиума по дисциплине «Управление проектами в сфере ИИ», завершающей блок дисциплины: применение методов управления качеством (стандартизация, функционально-стоимостной и функционально-физический анализ, диаграмма Парето), определение метрик качества ML-моделей и разработка плана управления качеством AI-продукта с последующей устной защитой.

## Состав

- **[kim_6_lab_task.md](kim_6_lab_task.md)** — полный текст КИМ для обучающихся;
- **[kim_6_lab_mark.md](kim_6_lab_mark.md)** — критерии оценивания;
- **[kim_6_lab_template.md](kim_6_lab_template.md)** — шаблон отчёта обучающегося;
- **[kim_6_lab_recom.md](kim_6_lab_recom.md)** — методические указания для преподавателя;
- **[kim_6_lab_var.md](kim_6_lab_var.md)** — варианты индивидуализации задания.

## Источник содержания

Комплект подготовлен на основе методов управления качеством проекта (стандартизация процессов, функционально-стоимостной и функционально-физический анализ, диаграмма Парето) и практики систем менеджмента качества, адаптированных к специфике разработки ИИ-систем: метрикам качества ML-моделей, мониторингу дрейфа данных и модели после внедрения. Комплект адаптирован к логике единого пакета ФОС, использованной в предыдущих работах цикла: терминология приведена к единому виду с приставкой «AI-» (AI-проект, AI-продукт, AI-команда), а план управления качеством опирается на методологию разработки (CRISP-DM/CRISP-ML(Q)) и распределение ролей, определённые в более ранних работах цикла.
