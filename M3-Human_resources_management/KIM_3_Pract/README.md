# Формирование AI-команды проекта

Папка содержит комплект документов для практической работы по дисциплине «Управление проектами» - «Формирование команды проекта» (диагностика ролевых предпочтений участников по методикам Р.М. Белбина, DISC, Р.О. Немова и С. Деллингера, ранжирование ролей и распределение функций внутри AI-команды).

КИМ содержит: паспорт, проверяемые результаты, ход выполнения, формат сдачи, оценку результатов, контрольные вопросы, использование ИИ-инструментов и рекомендации преподавателю.
