# M3. Управление человеческими ресурсами проекта (команда)

Папка содержит контрольно-измерительные материалы для модуля 3 дисциплины «Управление проектами в сфере ИИ».

Состав

KIM_2_Lab — контрольно-измерительные материалы (в виде лабораторной работы) для модуля 3, включающего следующие темы:

Разработка устава AI-проекта;
Применение Impact Mapping, User Story Mapping и WBS для определения содержания и границ проекта.
KIM_2_Pract — контрольно-измерительные материалы (в виде практического задания) по третьей части модуля 2, включающей следующие темы:

Анализ производственной задачи предприятия, определение целесообразности применения ИИ, постановка целей проекта;
Выявление требований заказчика, определение границ проекта, анализ стейкхолдеров, выбор состава работ.
Источник содержания
Образец подготовлен на основе рабочей программы дисциплины «Управление проектами в сфере ИИ» (раздел "Модуль 2. Управление содержанием проекта": инструменты целеполагания и декомпозиции в проектах; управление содержанием технологических проектов; разработка устава и содержания AI-проекта) и адаптирован к логике ФОС.
