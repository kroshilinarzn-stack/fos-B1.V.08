# М3. Формирование AI-команды, рабочего пространства и перечня работ AI-проекта

Папка содержит контрольно-измерительный материал (КИМ_3_Lab) для модуля 3 дисциплины «Управление проектами в сфере ИИ».

КИМ содержит: паспорт, проверяемые результаты, ход выполнения, формат сдачи, оценку результатов, контрольные вопросы, использование ИИ-инструментов и рекомендации преподавателю.
