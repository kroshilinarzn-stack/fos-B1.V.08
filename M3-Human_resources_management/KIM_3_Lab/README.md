# M3. Управление человеческими ресурсами в AI-проектах

**Папка содержит контрольно-измерительные материалы для модуля 3 дисциплины «Управление проектами в сфере ИИ».**

---

## Состав

**KIM_3_Lab** — контрольно-измерительные материалы (в виде лабораторной работы) к модулю 3, включающей следующие темы:

- Ролевая структура AI-проекта: специфика и ключевые позиции;
- Формирование кросс-функциональной команды AI-проекта;
- Управление гибридной командой «человек + ИИ-агенты»;
- Риски управления человеческими ресурсами в условиях высокой неопределённости.

**KIM_3_Pract** — контрольно-измерительные материалы (в виде практического задания) к модулю 3, включающей следующие темы:

- Разработка матрицы ответственности (RACI) и плана коммуникаций для AI-проекта;
- Применение инструментов мотивации, адаптированных к проектной деятельности в сфере ИИ;
- Использование технологий командообразования применительно к проектным группам;
- Планирование и подбор персонала для AI-проекта на разных стадиях его жизненного цикла;
- Взаимодействие с внутренними и внешними стейкхолдерами проекта в части управления командой;
- Оценка эффективности работы команды AI-проекта и разработка мероприятий по её развитию.
