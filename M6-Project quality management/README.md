# M6. Управление качеством проекта

Папка содержит образец полного КИМ для модуля 6 дисциплины «Управление проектами в сфере ИИ».

Назначение образца — показать единый формат оформления модульного КИМ в репозитории ФОС: паспорт, проверяемые результаты, ход выполнения, формат сдачи, рубрика, контрольные вопросы, использование ИИ-инструментов и рекомендации преподавателю.

## Состав

- `task.md` — полный текст КИМ для обучающихся;
- `rubric.md` — критерии оценивания и шкала 4/6/8/10;
- `student_report_template.md` — шаблон отчёта обучающегося;
- `teacher_notes.md` — методические указания для преподавателя;
- `variants.md` — варианты индивидуализации и усложнения задания.

## Источник содержания

Образец подготовлен на основе рабочей программы дисциплины «Управление проектами в сфере ИИ» (раздел «Модуль 6. Управление качеством проекта»: методы управления качеством в проектах; качество в технологических компаниях: стандарты и практика; план качества и метрики для AI-продукта) и адаптирован к логике ФОС.

## Примечание по форме контроля

Согласно РПД, форма контроля по модулю 6 — рубежный и текущий контроль (коллоквиум / лабораторная работа «план управления качеством»). В `task.md` и `rubric.md` учтён комбинированный характер контроля: письменный артефакт (план управления качеством) дополняется устной защитой (коллоквиум).
