# Модуль 5. Управление стоимостью проекта (EVM)

Папка содержит комплект документов для практического занятия по дисциплине «Управление проектами в сфере ИИ», формирующего компетенцию «Способен оценивать и прогнозировать экономические эффекты от внедрения ИИ».

## Состав

- **[kim_5_pract_task.md](kim_5_pract_task.md)** — полный текст КИМ для обучающихся;
- **[kim_5_pract_mark.md](kim_5_pract_mark.md)** — критерии оценивания;
- **[kim_5_lpract_template.md](kim_5_pract_template.md)** — шаблон отчёта обучающегося;
- **[kim_5_pract_recom.md](kim_5_pract_recom.md)** — методические указания для преподавателя;
- **[kim_5_pract_var.md](kim_5_pract_var.md)** — варианты индивидуализации задания.

## Источник содержания

Комплект подготовлен на основе методики освоенного объёма (Earned Value Management, EVM) для управления стоимостью проектов и адаптирован к логике единого пакета ФОС, использованной в предыдущих работах цикла: терминология приведена к единому виду с приставкой «AI-» (AI-проект, AI-команда, AI-роли), а в качестве исходных данных для расчёта показателей EVM используется перечень работ и этапов AI-проекта, сформированный в более ранних лабораторных работах цикла.
