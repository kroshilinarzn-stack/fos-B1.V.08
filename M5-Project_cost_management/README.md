# M5. Управление стоимостью проекта

Папка содержит образец полного КИМ для модуля 5 дисциплины «Управление проектами в сфере ИИ».

## Состав

**[KIM_5_Lab](KIM_5_Lab/README.md)** — контрольно-измерительные материалы (в виде лабораторной работы) к модулю 4, включающей следующие темы:
- разработка сетевой модели проекта;
- построение диаграммы Ганта с учётом зависимостей задач;
- расчёт критического пути и резервов времени.

## Источник содержания

Образец подготовлен на основе рабочей программы дисциплины «Управление проектами в сфере ИИ» (раздел «Модуль 5. Управление стоимостью проекта»: методика освоенного объёма и управление бюджетом; бюджетирование и контроль стоимости в ИИ-проектах; расчёт бюджета проекта через EVM) и адаптирован к логике ФОС.
