# Программа ДПО. 
** Преподавание современного ИИна топ-уровне: системное видение. Группа 40. РПД "Управление проектами в сфере ИИ

## 📌 О курсе

Дисциплина «Управление проектами в сфере ИИ» входит в цикл профессиональных модулей программы дополнительного профессионального образования (ДПО). Курс ориентирован на формирование у слушателей системного подхода к планированию, организации и контролю проектов по разработке и внедрению искусственного интеллекта.
