# Формирование AI-команды, рабочего пространства и перечня работ AI-проекта

Папка содержит комплект документов для объединённой лабораторной работы по дисциплине «Управление проектами», собранной из практического задания №2 «Создание рабочего пространства и распределение ролей» и практического задания №4 «Формирование перечня работ и определение логических связей».

## Состав

- `task.md` — полный текст лабораторной работы для обучающихся;
- `rubric.md` — критерии оценивания и шкала 4/6/8/10;
- `student_report_template.md` — шаблон отчёта AI-команды;
- `teacher_notes.md` — методические указания для преподавателя;
- `variants.md` — варианты индивидуализации и усложнения задания.

## Источник содержания

Комплект подготовлен на основе объединения практического задания №2 (распределение AI-ролей, рабочее пространство, RACI-матрица, тематика проектов) и практического задания №4 (WBS верхнего уровня, перечни работ по ролям, внутренние и внешние зависимости, проверка на циклы, тупики и согласованность оценок). Все роли, работы и артефакты приведены к единой терминологии с приставкой «AI-» (AI-проект, AI-команда, AI-роли, AI-продукт) и адаптированы к логике единого пакета ФОС.
