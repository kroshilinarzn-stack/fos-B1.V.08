# Формирование AI-команды, рабочего пространства и перечня работ AI-проекта

Папка содержит комплект документов для лабораторной работы по дисциплине «Управление проектами» и посвящена созданию рабочего пространства и распределению ролей, а также формированию перечня работ и определению логических связей.

## Состав

- `kim_3_lab_task.md` — полный текст лабораторной работы для обучающихся;
- `kim_3_lab_rubric.md` — критерии оценивания и шкала 4/6/8/10;
- `kim_3_lab_student_report_template.md` — шаблон отчёта AI-команды;
- `kim_3_lab_teacher_notes.md` — методические указания для преподавателя;
- `kim_3_lab_variants.md` — варианты индивидуализации и усложнения задания.

## Источник содержания

Все роли, работы и артефакты адаптированы к логике единого пакета ФОС.
