# Формирование AI-команды проекта

Папка содержит комплект документов для практической работы по дисциплине «Управление проектами» - «Формирование команды проекта» (диагностика ролевых предпочтений участников по методикам Р.М. Белбина, DISC, Р.О. Немова и С. Деллингера, ранжирование ролей и распределение функций внутри команды).

## Состав

- `kim_3_practice_task.md` — полный текст практической работы для обучающихся;
- `kim_3_practice_rubric.md` — критерии оценивания и шкала 4/6/8/10;
- `kim_3_practice_student_report_template.md` — шаблон отчёта AI-команды;
- `kim_3_practice_teacher_notes.md` — методические указания для преподавателя;
- `kim_3_practice_variants.md` — варианты индивидуализации и усложнения задания.

## Источник содержания

Комплект подготовлен на основе разработанной РПД.
