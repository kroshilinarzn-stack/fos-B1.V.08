# Варианты индивидуализации задания

## Вариант A. Базовый

Пройти все 4 методики, построить гистограммы по AI-команде, составить индивидуальные ранжированные профили и сформировать итоговую матрицу распределения ролей без глубокого анализа конфликтов.

## Вариант B. Разрешение конфликтов ролей

Смоделировать ситуацию, когда двое участников AI-команды претендуют на одну и ту же основную роль (например, оба — сильные Треугольники по Деллингеру и D по DISC). Подробно описать процесс и критерии разрешения конфликта.

## Вариант C. Команда без выраженного лидера

Проанализировать ситуацию, когда ни у одного участника AI-команды нет высокого балла по ролям блока А (лидерство и координация). Предложить способ покрытия дефицита за счёт поддерживающих ролей (Ранг 2).

## Вариант D. Расширенная AI-команда

Выполнить работу для AI-команды из 7–8 человек вместо 5, адаптировав правило функциональной полноты (допускается несколько участников на один функциональный блок, но все блоки должны быть покрыты).

## Вариант E. Сравнительный анализ методик

Провести сопоставление результатов всех 4 методик для каждого участника, выявить случаи расхождения (например, роль лидера по Немову, но фигура «Круг» по Деллингеру) и предложить интерпретацию таких расхождений.

## Вариант F. Оценка устойчивости распределения ролей

Оценить, как изменится итоговая матрица распределения ролей, если один участник AI-команды будет заменён новым (с собственными результатами тестов), и какие функциональные блоки окажутся под риском.
