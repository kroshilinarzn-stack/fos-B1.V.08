# Формирование AI-команды проекта

Папка содержит комплект документов для практической работы по дисциплине «Управление проектами», подготовленной на основе лабораторной работы №1 «Формирование команды проекта» (диагностика ролевых предпочтений участников по методикам Р.М. Белбина, DISC, Р.О. Немова и С. Деллингера, ранжирование ролей и распределение функций внутри команды).

## Состав

- `task.md` — полный текст практической работы для обучающихся;
- `rubric.md` — критерии оценивания и шкала 4/6/8/10;
- `student_report_template.md` — шаблон отчёта AI-команды;
- `teacher_notes.md` — методические указания для преподавателя;
- `variants.md` — варианты индивидуализации и усложнения задания.

## Источник содержания

Комплект подготовлен на основе лабораторной работы №1 «Формирование команды проекта» и адаптирован в формат практической работы: терминология приведена к единому виду с приставкой «AI-» (AI-команда, AI-роли, AI-проект), сохранены исходные диагностические методики (Приложения 1–4) и правила ранжирования и распределения ролей. Полные тексты методик прилагаются отдельным файлом-приложением.
