# Оценивание КИМ 3

Оценивание проводится по единой 10-балльной шкале ФОС. Перевод результата в БРС выполняется не умножением на 10, а по весу данного КИМ, который фиксируется в `docs/evaluation_policy.md`.

## 1. Критерии оценивания

| Критерий | Баллы |
|---|---:|
| Анализ кейса и корректная постановка задачи по формированию команды | 1.0 |
| Проектирование организационной структуры команды и распределение ролей | 1.5 |
| Разработка и обоснование матрицы RACI | 2.0 |
| Анализ конфликтов интересов и предложенные способы их разрешения | 1.5 |
| Описание принципов лидерства и самоорганизации в R&D-команде | 1.5 |
| Описание модели кросс-функционального взаимодействия | 1.0 |
| Глубина и обоснованность дополнительного исследования / выводов | 1.0 |
| Качество оформления отчёта и артефактов (RACI-матрица, схема структуры) | 0.5 |
| **Итого** | **10.0** |

## 2. Уровни достижения результата

| Уровень | Баллы | Описание |
|---|---:|---|
| Не достигнут | 0–3 | Задание не выполнено или выполнено фрагментарно; отсутствует рабочая RACI-матрица; роли не определены. |
| Базовый | 4–5 | Определена структура команды и роли, построена простая RACI-матрица без достаточного обоснования; анализ конфликтов и лидерства поверхностный. |
| Средний | 6–7 | RACI-матрица построена корректно и обоснована для нескольких этапов; описаны типичные конфликты и способы их разрешения; есть базовое описание лидерства и кросс-функционального взаимодействия. |
| Высокий | 8–9 | Все элементы задания выполнены полно и содержательно; RACI-матрица детализирована; анализ конфликтов и лидерства опирается на конкретику кейса. |
| Продвинутый | 10 | Работа демонстрирует системное понимание управления человеческими ресурсами AI-проекта; выполнено содержательное дополнительное исследование; предложены практико-ориентированные рекомендации. |

## 3. Пересчёт в БРС

Вес КИМ в балльно-рейтинговой системе определяется в документе `docs/evaluation_policy.md`.

```text
балл_БРС = (балл_КИМ / 10) × вес_КИМ
```

Например, если вес КИМ в БРС составляет 4 балла, а студент получил 7 из 10 по данной рубрике, итоговый вклад КИМ равен 2,8 балла.

## 4. Типовые основания для снижения баллов

- роли команды не привязаны к специфике конкретного AI-проекта (шаблонное перечисление без адаптации к кейсу);
- матрица RACI построена без обоснования, либо содержит противоречия (например, несколько «Accountable» на один этап);
- отсутствует анализ конфликтов интересов или он ограничивается общими фразами;
- принципы лидерства и самоорганизации описаны абстрактно, без связи с этапами развития команды;
- не описано кросс-функциональное взаимодействие между data science, engineering и бизнес-подразделениями;
- отчёт не содержит визуализации организационной структуры;
- не указано использование генеративного ИИ при подготовке текста.
