**5. СИСТЕМА ОЦЕНКИ РЕЗУЛЬТАТОВ ОБУЧЕНИЯ**

**Формы контроля и КИМ по модулям дисциплины**

  ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  **Модуль**                                                  **Компетенции**          **Дескриптор освоения**                                                                                                                                                                                                                                                                           **Форма контроля**   **Вид КИМ**
  ----------------------------------------------------------- ------------------------ ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- -------------------- ---------------------------------------------------------------------------------------
  Модуль 1. Проекты и среда управления                        LC-1; SS-2               Определяет и формализует проблему предметной области, решение которой требует применения ИИ; формализует особенности проектов в сфере ИИ. Понимает общую цель команды, участвует в обсуждении задач планирования и реализации проекта.                                                            Текущий              Опрос / мини-эссе; практическое задание

  Модуль 2. Управление содержанием проекта                    LC-1                     Проводит анализ требований к системе ИИ, оценивает и разрабатывает технические требования на основе формализованной постановки задачи. Формализует бизнес-цель заказчика, выявляет заинтересованные стороны и требования; разрабатывает ТЗ на систему ИИ с учётом специфики предметной области.   Текущий              Практическое задание; лабораторная работа (устав проекта, WBS)

  Модуль 3. Управление человеческими ресурсами проекта        LC-4; SS-2               Обеспечивает взаимодействие всех участников проекта. Осуществляет коммуникации между аналитиками данных, инженерами, разработчиками и руководителями подразделений для масштабирования ИИ-инициатив.                                                                                              Текущий              Практическое задание; лабораторная работа (матрица RACI)

  Модуль 4. Управление сроками проекта                        LC-4                     Осуществляет ведение проекта в области ИИ, планирование и контроль задач, оценку ресурсов. Подбирает методологию (CRISP-DM, CRISP-ML(Q)) и организует процесс разработки.                                                                                                                         Текущий              Практическое задание; лабораторная работа (сетевой график, диаграмма Ганта)

  Модуль 5. Управление стоимостью проекта                     LC-7                     Знает ключевые технико-экономические показатели внедрения ИИ. Строит расчётные модели для оценки технико-экономических показателей ИИ и применяет их для прогнозирования.                                                                                                                         Текущий              Практическое задание; лабораторная работа (бюджет по методике освоенного объёма)

  Модуль 6. Управление качеством проекта                      LC-4                     Подбирает методологию разработки системы ИИ. Организует процесс контроля качества AI-продукта, применяет инструменты приоритизации дефектов и метрики качества ML-моделей.                                                                                                                        Рубежный; текущий    Коллоквиум / практическое задание; лабораторная работа (план управления качеством)

  Модуль 7. Управление рисками и взаимодействием в проекте    LC-7                     Знает ключевые риски внедрения ИИ. Строит расчётные оценки экономического ущерба и разрабатывает модели экономических эффектов внедрения ИИ.                                                                                                                                                      Текущий              Практическое задание; лабораторная работа (реестр рисков TOP-10)

  Модуль 8. Методы и подходы к управлению проектами           LC-4                     Организует процесс разработки системы ИИ по гибкой методологии. Подбирает методологию управления ИИ-проектом и практически реализует её.                                                                                                                                                          Рубежный; текущий    Коллоквиум / решение практического кейса; лабораторная работа (спринт, ретроспектива)

  Модуль 9. Программный инструментарий управления проектами   LC-4                     Демонстрирует эффективное владение инструментами коммуникаций, проектного управления и контроля изменений.                                                                                                                                                                                        Текущий              Практическое задание

  Промежуточная аттестация                                    LC-1, LC-4, LC-7, SS-2   Итоговая проверка сформированности компетенций дисциплины на уровне «средний».                                                                                                                                                                                                                    Промежуточный        Билеты + практико-ориентированное задание
