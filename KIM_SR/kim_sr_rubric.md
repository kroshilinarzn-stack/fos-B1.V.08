# Рубрика оценивания КИМ СР

Оценивание проводится по единой 10-балльной шкале ФОС. Перевод результата в БРС выполняется не умножением на 10, а по весу данного КИМ, который фиксируется в `docs/evaluation_policy.md`.

## 1. Критерии оценивания

| Критерий | Баллы |
|---|---:|
| Корректность постановки проблемы и связь темы с содержанием модуля | 1.5 |
| Полнота и качество обзора источников по теме (не менее 3 источников) | 2.5 |
| Глубина анализа применимости подхода к специфике ИИ-проектов | 2.5 |
| Обоснованность и самостоятельность собственного вывода | 2.0 |
| Корректность оформления ссылок и списка литературы | 1.0 |
| Качество оформления и структурированность текста эссе | 0.5 |
| **Итого** | **10.0** |

## 2. Уровни достижения результата

| Уровень | Баллы | Описание |
|---|---:|---|
| Не достигнут | 0–3 | Эссе не выполнено или выполнено фрагментарно; источники не указаны или не использованы; текст представляет собой пересказ без анализа. |
| Базовый | 4–5 | Тема раскрыта поверхностно; использовано менее 3 источников; анализ применимости к ИИ-проектам минимален; собственный вывод отсутствует или формален. |
| Средний | 6–7 | Тема раскрыта, использовано не менее 3 источников; есть анализ применимости к специфике ИИ-проектов; вывод присутствует, но недостаточно обоснован. |
| Высокий | 8–9 | Тема раскрыта полно и содержательно; источники разнообразны и корректно процитированы; анализ применимости к ИИ-проектам глубокий; вывод обоснован и самостоятелен. |
| Продвинутый | 10 | Эссе демонстрирует самостоятельное критическое мышление; использованы источники сверх минимального требования, включая специализированные по ИИ-проектам; вывод содержит практико-ориентированные рекомендации. |

## 3. Пересчёт в БРС

Вес КИМ в балльно-рейтинговой системе определяется в документе `docs/evaluation_policy.md`.

```text
балл_БРС = (балл_КИМ / 10) × вес_КИМ
```

Например, если вес КИМ в БРС составляет 2 балла, а студент получил 8 из 10 по данной рубрике, итоговый вклад КИМ равен 1,6 балла.

## 4. Типовые основания для снижения баллов

- эссе представляет собой пересказ одного источника без сравнения точек зрения;
- используется менее 3 источников или источники не относятся к теме;
- отсутствует анализ применимости темы к специфике проектов в сфере ИИ (текст носит общий характер, не связанный с ИИ-контекстом);
- вывод отсутствует либо сводится к общей фразе без обоснования;
- ссылки на источники в тексте отсутствуют или не соответствуют списку литературы;
- объём эссе существенно меньше рекомендованного (менее 2 страниц по существу);
- не указано использование генеративного ИИ при подготовке текста, либо текст содержит признаки полностью нередактированной генерации.
