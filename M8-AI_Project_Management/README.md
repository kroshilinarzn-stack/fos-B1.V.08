# M8. Управление проектами в сфере искусственного интеллекта

Папка содержит контрольно-измерительные материалы для модуля 8 дисциплины «Управление проектами в сфере ИИ».

## Состав

**[KIM_8_Lab](KIM_8_Lab/)** — контрольно-измерительные материалы (**в виде лабораторной работы**) по модулю 8, включающие следующие темы:

- гибкие методологии управления проектами в ИИ-контексте;
- выбор методологии для AI-проекта; 
- планирование спринта и Scrum-церемонии; 
- инструментальная среда проектного менеджмента

**[KIM_8_Colloq](KIM_8_Colloq/)** —  **коллоквиум** по  7 и 8 модулям дисциплины: банк теоретических вопросов («дайте определение понятия») и проблемных ситуаций (кейсов) в формате Moodle XML, разбитый на две тематические подгруппы.

## Источник содержания

Образец подготовлен на основе рабочей программы дисциплины «Управление проектами в сфере ИИ» (раздел «Модуль 8. Гибкие методологии управления проектами в ИИ-контексте; выбор методологии для AI-проекта; планирование спринта и Scrum-церемонии") и адаптирован к логике ФОС.
