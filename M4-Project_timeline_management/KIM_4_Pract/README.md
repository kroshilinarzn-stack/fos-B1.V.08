# Подбор методологии разработки AI-проекта (CRISP-DM / CRISP-ML(Q))

Папка содержит комплект документов для практического занятия по дисциплине «Управление проектами», формирующего компетенцию «Осуществляет ведение проекта в области ИИ, планирование и контроль задач, оценку ресурсов. Подбирает методологию (CRISP-DM, CRISP-ML(Q)) и организует процесс разработки».

## Состав

- `kim_4_practice_task.md` — полный текст практического занятия для обучающихся;
- `kim_4_practice_rubric.md` — критерии оценивания и шкала 4/6/8/10;
- `kim_4_practice_student_report_template.md` — шаблон отчёта обучающегося;
- `kim_4_practice_teacher_notes.md` — методические указания для преподавателя;
- `kim_4_practice_variants.md` — варианты индивидуализации и усложнения задания.

## Источник содержания

Комплект подготовлен на основе методологий CRISP-DM и CRISP-ML(Q) для управления проектами машинного обучения и адаптирован к логике единого пакета ФОС, использованной в предыдущих работах цикла: терминология приведена к единому виду с приставкой «AI-» (AI-проект, AI-команда, AI-роли), а в качестве исходных данных для адаптации этапов методологии используется перечень работ AI-проекта, сформированный в более ранних лабораторных работах цикла.
