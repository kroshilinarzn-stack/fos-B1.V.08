# M4. Управление сроками проекта

Папка содержит образец полного КИМ для модуля 4 дисциплины «Управление проектами в сфере ИИ».

## Состав

**[KIM_4_Lab](KIM_3_Lab/README.md)** — контрольно-измерительные материалы (в виде лабораторной работы) к модулю 3, включающей следующие темы:
- разработка сетевой модели проекта;
- построение диаграммы Ганта с учётом зависимостей задач;
- расчёт критического пути и резервов времени.

**[KIM_4_Pract](KIM_4_Pract/README.md)** — контрольно-измерительные материалы (в виде лабораторной работы) к модулю 3, включающей следующие темы:
- корректировка календарного плана при изменении требований;
- корректировка календарного плана при появлении новых данных;
- корректировка календарного плана при изменении результатов обучения моделей.
  
## Источник содержания

Образец подготовлен на основе рабочей программы дисциплины «Управление проектами в сфере ИИ» (раздел «Модуль 4. Управление сроками проекта»: календарное планирование и сетевые модели в проектах; управление сроками проекта в условиях неопределённости; сетевое планирование проекта; планирование сроков реализации AI-проекта) и адаптирован к логике ФОС.
