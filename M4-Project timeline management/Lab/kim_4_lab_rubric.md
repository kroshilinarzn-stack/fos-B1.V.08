# Рубрика оценивания практической работы «Формирование AI-команды проекта»

Оценивание проводится по единой 10-балльной шкале ФОС. Перевод результата в БРС выполняется не умножением на 10, а по весу данной работы, который фиксируется в `docs/evaluation_policy.md`.

## 1. Критерии оценивания

| Критерий | Баллы |
|---|---:|
| Полнота и корректность прохождения всех 4 методик каждым участником AI-команды | 1.5 |
| Корректность обработки результатов и заполнения ключей-таблиц по каждой методике | 1.5 |
| Построение и корректная интерпретация гистограмм по AI-команде | 1.5 |
| Качество индивидуальных ранжированных ролевых профилей | 1.5 |
| Корректность итоговой матрицы распределения ролей в AI-команде | 1.5 |
| Функциональная полнота AI-команды (представлены все 5 блоков) | 1.0 |
| Обоснованность принятых решений, разрешение конфликтов ролей | 1.0 |
| Оформление отчёта и соблюдение требований | 0.5 |
| **Итого** | **10.0** |

## 2. Уровни достижения результата

| Уровень | Баллы | Описание |
|---|---:|---|
| Не достигнут | 0–3 | Тесты пройдены не всеми участниками или результаты не обработаны; итоговая матрица ролей отсутствует. |
| Базовый | 4–5 | Тесты пройдены и обработаны, построены отдельные гистограммы, но ранжированные профили неполные, итоговая матрица не обоснована. |
| Средний | 6–7 | Построены индивидуальные ранжированные профили по всем участникам, сформирована итоговая матрица распределения ролей, представлено базовое обоснование. |
| Высокий | 8–9 | Все методики корректно обработаны, конфликты ролей разрешены и аргументированы, обеспечена функциональная полнота AI-команды. |
| Продвинутый | 10 | Отчёт полностью соответствует требованиям оформления, обоснование решений опирается на количественные данные по всем методикам, дан содержательный вывод о готовности AI-команды к AI-проекту. |

## 3. Пересчёт в БРС

Вес работы в балльно-рейтинговой системе определяется в документе `docs/evaluation_policy.md`.

```text
балл_БРС = (балл_работы / 10) × вес_работы
```

## 4. Типовые основания для снижения баллов

- не все участники AI-команды прошли все 4 методики;
- нарушено правило распределения баллов внутри разделов теста (например, сумма баллов в разделе Белбина не равна 10);
- гистограммы не построены или не подписаны;
- индивидуальный ранжированный профиль не учитывает результаты одной или нескольких методик;
- итоговая матрица распределения ролей не обеспечивает функциональную полноту AI-команды (отсутствует блок);
- конфликт ролей не зафиксирован или разрешён без обоснования;
- выводы не подкреплены конкретными числовыми значениями тестов;
- не указано использование генеративного ИИ при подготовке отчёта.
