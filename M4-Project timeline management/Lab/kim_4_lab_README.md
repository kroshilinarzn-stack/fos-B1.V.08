# Разработка сетевой модели проекта

Папка содержит комплект документов для лабораторной работы по дисциплине «Управление проектами», продолжающей цикл работ по управлению AI-проектом: построение сетевого графика AI-проекта, расчёт временных параметров, определение критического пути и построение диаграммы Ганта на основе перечня работ, сформированного в предыдущей лабораторной работе.

## Состав

- `task.md` — полный текст лабораторной работы для обучающихся;
- `rubric.md` — критерии оценивания и шкала 4/6/8/10;
- `student_report_template.md` — шаблон отчёта обучающегося;
- `teacher_notes.md` — методические указания для преподавателя;
- `variants.md` — варианты индивидуализации и усложнения задания.

## Источник содержания

Комплект подготовлен на основе методики сетевого планирования (расчёт ES/EF/LS/LF, полного и свободного резервов времени, определение критического пути, построение диаграммы Ганта) и адаптирован к логике единого пакета ФОС, использованной в предыдущих работах цикла: терминология приведена к единому виду с приставкой «AI-» (AI-проект, AI-команда, AI-роли), а исходными данными служит перечень работ, сформированный в лабораторной работе «Формирование AI-команды, рабочего пространства и перечня работ AI-проекта».
